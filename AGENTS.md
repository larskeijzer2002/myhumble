## UX regels
- Homepage is de startpagina
- Quiz opent alleen via CTA-knoppen
- Na quiz volgt pakketoverzicht
- Klik op pakketnaam opent verborgen detailpagina
- Klik op "Kies dit pakket" gaat naar Stripe
- Privacy moet een echte overlay/pagina blijven
- Footer moet premium ogen

## Tech regels
- React + Tailwind
- Houd code modulair
- Gebruik kleine herbruikbare componenten
- Vermijd duplicatie
- Houd styling consistent
- Behoud bestaande functionaliteit tenzij expliciet verbeterd
